# Event Manager application

This project is a web application - Event Manager.

## Used technologies

- React (Next.js)
- TypeScript
- TailwindCSS

## Installation
1. Clone the repo
```bash
git clone https://github.com/akgane/event-manager.git
```
2. Install NPM packages
```bash
npm install
```
3. Run the app using
```bash
npm run start
```
Server will run on 
```bash
localhost:3000
```
